## 🛡️ Система безопасности

| Защита | Описание |
|--------|----------|
| VM Detection | Работает только в виртуальных машинах |
| Safe Mode | При любой ошибке → откат, система грузится нормально |
| Sanity Checks | Проверка всех указателей и размеров |
| Rollback | Автоматический откат при сбое |
| Logging | Полное логирование для отладки |

## ⚠️ Требования

- Intel CPU с поддержкой VT-x
- UEFI firmware
- Windows 10/11 x64
- Тестировать ТОЛЬКО в VM!

## 🚀 Использование

1. Скомпилировать bootkit и hypervisor
2. Установить bootkit в ESP
3. Загрузить Windows (hypervisor запустится автоматически)
4. Запустить usermode overlay
5. INSERT = меню
